cshdj-soundcloud
================

SoundCloud plugin for CSH DJ.

Configuration
=============

* Register for a client ID and secret [here](http://soundcloud.com/you/apps).
* Put it in the plugin config: 
```
configuration: {
   //...
   'cshdj-soundcloud': {
      auth: {
	     client_id: 'YOUR_CLIENT_ID',
		 client_secret: 'YOUR_CLIENT_SECRET'
	  }
   }
}
```
